# 🐍 Snake Arcade Game

A cyber-styled Snake game built using **HTML5 Canvas, CSS and JavaScript**.

## 🎮 Play the Game
https://SkyonixTheKing23.github.io/snake-game/

## Features
- Neon cyber UI
- Particle effects
- Level system
- Score tracking
- Mobile swipe controls

## Controls
Arrow Keys or WASD

## Technologies
- HTML5 Canvas
- CSS
- JavaScript
